# Response Examples for Various Thirdparty APIs

This file contains response examples that have been deemed by me (the user) to be important references for how you (the agent) implement the dashboards. While I have pasted the endpoint URLs below, remember to use the relevant Vercel SWR hook that has been generated by Orval. Also, remember to implement the Next.JS proxy for it if unimplemented.
